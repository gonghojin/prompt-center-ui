# 프롬프트 좋아요 기능 구현 계획

## 1. API 엔드포인트 설계

### 1.1 좋아요 관련 API

- `POST /api/prompts/{promptId}/like` - 좋아요 추가
- `DELETE /api/prompts/{promptId}/like` - 좋아요 삭제
- `GET /api/prompts/{promptId}/likes` - 좋아요 목록 조회
- `GET /api/prompts/{promptId}/likes/count` - 좋아요 수 조회

### 1.2 응답 형식

```typescript
interface LikeResponse {
  success: boolean;
  data: {
    promptId: string;
    likeCount: number;
    isLiked: boolean;
  };
}
```

## 2. 프론트엔드 구현 계획

### 2.1 컴포넌트 구현

1. **LikeButton 컴포넌트**
    - 좋아요 토글 버튼
    - 좋아요 수 표시
    - 애니메이션 효과
    - 로딩 상태 처리

2. **LikeCount 컴포넌트**
    - 좋아요 수 표시
    - 실시간 업데이트

3. **LikeList 모달**
    - 좋아요한 사용자 목록
    - 사용자 프로필 미리보기
    - 페이지네이션

### 2.2 페이지별 구현

1. **프롬프트 탐색 페이지 (`/prompts`)**
    - 프롬프트 카드에 좋아요 버튼 추가
    - 좋아요 수 표시
    - 좋아요순 정렬 옵션 추가

2. **내 프롬프트 관리 페이지 (`/my-prompts`)**
    - 좋아요한 프롬프트 탭 추가
    - 좋아요 통계 섹션 구현
    - 좋아요 추이 그래프 구현

3. **프롬프트 상세 페이지 (`/prompts/[id]`)**
    - 상세 정보에 좋아요 버튼 추가
    - 좋아요한 사용자 목록 모달
    - 좋아요 알림 설정 옵션
